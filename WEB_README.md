# Flutter Web Design System

Веб-приложение для демонстрации дизайн-токенов и компонентов.

## Запуск веб-приложения

Для запуска веб-версии используйте:

```bash
fvm flutter run -d chrome --target=lib/main_web.dart
```

Или для сборки:

```bash
fvm flutter build web --target=lib/main_web.dart
```

## Структура веб-приложения

- `lib/main_web.dart` - точка входа для веб-приложения
- `lib/ui_web/app_web.dart` - главный виджет приложения с навигацией
- `lib/ui_web/pages/tokens_page.dart` - страница со списком всех дизайн-токенов
- `lib/ui_web/pages/components_page.dart` - интерактивный редактор компонентов

## Функционал

### Design Tokens
- Отображение всех цветов с hex-кодами (включая альфа-канал)
- Список всех значений отступов (spacing)
- Параметры типографии (шрифты, размеры, веса)
- Токены темы (text, border цвета)

### Components
- Интерактивный редактор для UiButton
- Настройка всех параметров:
  - Текст кнопки
  - Иконки (слева/справа)
  - Состояние (default, hover, disabled)
  - Стиль (main, minor)
  - Тип (accent, neutral, success, critical)
  - Размер (M, S)
  - Форма (squircle, circle)
- Предпросмотр на экране смартфона в реальном времени
- Переключение светлой/темной темы

## Особенности

- Отдельный entry point для веб-версии
- Использование Material Design для веб-интерфейса
- Адаптивный дизайн с боковой панелью навигации
- Интерактивный предпросмотр компонентов
